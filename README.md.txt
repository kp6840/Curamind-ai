# CuraMind AI

## Project Overview
CuraMind AI is an AI-powered healthcare platform for medical image analysis and telehealth.

 Key Features

###  Patient Module
- Secure user registration and login
- Upload medical images (X-ray, MRI, etc.)
- View AI-generated diagnostic reports
- Book and manage appointments
- Access medical history

###  Doctor Module
- View assigned patient cases
- Analyze uploaded medical images
- Review AI predictions (heatmaps, anomaly detection)
- Provide diagnosis and feedback
- Manage appointments

###  Admin Module
- Manage users (patients & doctors)
- Monitor system activities
- Maintain data integrity and security

###  AI Integration
- Image processing using Computer Vision models
- Detection of anomalies in medical images
- Generation of diagnostic insights


## Tech Stack
- Backend: Django
- Database: PostgreSQL
- AI: PyTorch
- Frontend: HTML/CSS/JS
